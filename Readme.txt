 # 🛡️ Cyber Attack Visualizer

A simple, interactive web-based simulator that visually demonstrates two common cyber attacks — **SQL Injection** and **Phishing** — using HTML5 Canvas and JavaScript.

![preview](preveiw.gif)

---

## 🚀 Features

- 🎯 Simulates **SQL Injection** and **Phishing** visually
- 🔊 Plays alert sound on successful attack
- 🎨 Smooth animations with HTML Canvas
- 🌙 Dark Mode UI
- 📱 Responsive layout for all screen sizes

---

## 🧰 Tech Stack

- **HTML5 + CSS3**
- **JavaScript**
- **Canvas API**
- Optional: [anime.js](https://animejs.com/) for advanced animations

---

## 🎮 How to Run

1. **Clone this repo**:
   ```bash
   git clone https://github.com/BhumiShriwas/cyber-attack-visualizer.git
   cd cyber-attack-visualizer



Add your sound:
Place a file named alert.mp3 in the project folder.

Open index.html in your browser.

🔊 Sound Setup
Rename your alert sound file to alert.mp3.

Download free alert sounds:

https://mixkit.co/free-sound-effects/

https://freesound.org

🌐 Deploy on GitHub Pages
Push to your GitHub repo.

Go to Settings → Pages.

Select main as branch and /root as folder.

Access your site at:
https://BhumiShriwas.github.io/cyber-attack-visualizer/
